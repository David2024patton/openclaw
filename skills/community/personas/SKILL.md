## Slash Commands

Use these commands any time for fast, explicit control.

**Activate a persona:**
```
/persona dev
/persona "Chef Marco"
```

**List personas:**
```
/persona list
/personas
```

**Exit current persona:**
```
/persona exit
```

**Create a new persona:**
```
/persona create "Game Master"
```
